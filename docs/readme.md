### 세팅
- learn.bat 작성한다.
- root폴더에서 main.py외에 필요한 py파일들을 만들어서 특정chapter를 학습한다.
- 학습 종료 후 아래와 같은 커맨드를 통해 학습내용을 정리한다.
    ```shell
    ./learn.bat [책_이름] [chapter명]
    ```
